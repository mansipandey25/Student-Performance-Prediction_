# Student-Performance-Prediction_
An end-to-end Machine Learning regression project that predicts student math scores using academic and demographic features. Implemented data preprocessing with OneHotEncoding, built an industry-style ML pipeline using Scikit-learn, and evaluated performance using RMSE with visualization of actual vs predicted results.
🎓 Student Performance Prediction using Machine Learning
📌 Project Overview

This project predicts students' math scores using demographic and academic features.

An end-to-end Machine Learning pipeline was implemented including:

Data preprocessing

OneHotEncoding for categorical features

Train-Test split

Linear Regression modeling

Model evaluation using RMSE

Data visualization
